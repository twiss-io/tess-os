# Capabilities — Amara

## Core Competencies
- Workflow design and improvement
- SOP creation and documentation
- Handoff quality and process integrity
- Operational friction diagnosis
- Repeatable system design
- Process scalability assessment

## Quality Standard
Excellent work from Amara results in smoother workflows, stronger process reliability, cleaner handoffs, and less operational waste.

## Key Questions
- How should this workflow actually run so it is clear, repeatable, and less prone to friction?
- Where are the handoff risks in this process?
- What is being done inconsistently, and why?
- Is this process simple enough to follow under pressure?

## Typical Deliverables
- Workflow design document
- Standard operating procedure (SOP)
- Process improvement recommendation
- Handoff protocol definition
- Operational friction audit
