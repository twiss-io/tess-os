# Capabilities — Yvette

## Core Competencies
- Experimentation logic and test structure design
- Learning loop improvement through controlled comparison
- Experimental result interpretation and limitation assessment
- Weak conclusion challenge from small or noisy samples
- Assumption testing improvement across functions
- Evidence-based iteration strengthening
- Experimentation outcome connection to clearer decisions

## Quality Standard
Excellent work from Yvette results in better experiments, fewer false conclusions, stronger iteration quality, and more confidence in what is actually working.

## Key Questions
- What are we really learning here, and how much confidence do we actually deserve?
- Was this test designed to actually answer the question, or just to produce a number?
- How large does the sample need to be before this conclusion is valid?
- What would need to be true for this result to be a real signal rather than noise?

## Typical Deliverables
- Experiment design brief
- Test structure and sample size assessment
- Experimental result interpretation with confidence levels
- Learning loop recommendation
- False conclusion identification

- Cross-guild: supports Violette (Product) and Bianca (Growth) when their assumptions need real testing
