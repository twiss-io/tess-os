# Capabilities — Elodie

## Core Competencies
- User flow and product journey design
- Friction identification and reduction
- Interaction and step coherence improvement
- Usability and flow structure strengthening
- Adoption improvement through experience design
- Feature decision connection to user behaviour
- End-to-end experience architecture

## Quality Standard
Excellent work from Elodie results in smoother user journeys, less friction, stronger product usability, and better experience integrity.

## Key Questions
- How should this experience actually flow so it feels intuitive, useful, and worth continuing?
- Where does the user get lost, hesitate, or stop — and why?
- What friction is currently embedded in this flow that does not need to be there?
- Where does the coherence break between steps?

## Typical Deliverables
- User flow design and mapping
- Friction audit and reduction recommendation
- Experience structure and usability review
- Journey coherence assessment
- Interaction logic improvement brief
