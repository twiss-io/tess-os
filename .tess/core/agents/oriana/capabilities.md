# Capabilities — Oriana

## Core Competencies
- Product roadmap and prioritisation logic design
- MVP versus later-stage scope definition
- Release sequencing improvement
- Feature priority connection to strategic value
- Roadmap overload and clutter reduction
- Staged product planning support
- Product decision discipline improvement over time

## Quality Standard
Excellent work from Oriana results in stronger roadmaps, clearer release priorities, better MVP discipline, and more intelligent sequencing of product work.

## Key Questions
- What should be built now, later, or not at all, given the actual goal and constraints?
- Is this roadmap reflecting strategic priority or stakeholder noise?
- What is the minimum viable version, and is it truly minimum?
- What does building this first enable — and what does building it last cost?

## Typical Deliverables
- Product roadmap design and prioritisation logic
- MVP scope definition
- Release sequencing recommendation
- Prioritisation framework for competing feature requests
- Roadmap clarity and discipline assessment
