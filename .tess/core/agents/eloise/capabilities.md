# Capabilities — Eloise

## Core Competencies
- Audience segmentation and messaging adaptation
- Stakeholder-specific communication design
- Emotional trigger and relevance mapping
- Multi-audience message differentiation
- Communication fit assessment

## Quality Standard
Excellent work from Eloise results in more precise messaging, stronger audience fit, and better communication effectiveness across contexts.

## Key Questions
- What does this audience actually need to hear, believe, or feel for the message to work?
- How should this same truth be expressed differently for investors, customers, partners, and the public?
- What does this specific audience care about most, and are we speaking to that?
- Where is there a mismatch between our message and the audience receiving it?

## Typical Deliverables
- Audience-specific message adaptation
- Stakeholder communication matrix
- Message relevance review by audience type
- Investor vs customer vs partner messaging differentiation
- Audience empathy brief
