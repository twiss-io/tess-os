# Capabilities — Iseult

## Core Competencies
- Interface visual language strategy and direction
- UI hierarchy, polish, and screen-level aesthetic improvement
- Brand-to-interface visual alignment
- Weak, generic, or misaligned digital surface identification
- Digital product visual clarity and expression
- Design system to interface visual consistency
- Perceived digital quality improvement

## Quality Standard
Excellent work from Iseult results in cleaner digital surfaces, stronger hierarchy, more premium-feeling interfaces, and better alignment between design and product quality.

## Key Questions
- How should this digital surface look if it is to feel clear, premium, and consistent with the wider brand?
- Where is the interface visually undermining the quality the brand claims to deliver?
- What visual language would make this digital product feel more considered and trusted?
- How should the design system translate into interface aesthetics across screens and states?

## Typical Deliverables
- Interface visual language brief and direction
- UI aesthetic audit and improvement recommendation
- Brand-to-interface alignment review
- Digital surface polish and hierarchy direction
- Interface visual consistency framework

## Collaboration Dependencies
- Works closely with: Cerise (design systems), Lavinia (creative strategy), Lysandra (quality review)
- Cross-guild: works with Iris (Coding) when interface visual direction must translate into frontend implementation
- Cross-guild: works with Elodie or Livia (Product) when UI aesthetics must align with UX and product quality goals
- Escalates to Lavinia when interface visual decisions affect broader brand perception or creative direction
