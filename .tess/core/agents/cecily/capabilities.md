# Capabilities — Cecily

## Core Competencies
- Transaction framing and strategic justification assessment
- Deal structure logic guidance
- Transaction-to-strategic-outcome connection
- Major value creation and destruction driver identification
- Leadership deal trade-off support
- Transaction posture setting before terms negotiation
- High-level deal direction and go/no-go judgment

## Quality Standard
Excellent work from Cecily results in smarter transaction direction, stronger structure, better deal judgment, and fewer costly strategic mistakes.

## Key Questions
- Should this deal happen at all, and if so, what structure best protects value while advancing the real objective?
- What is the strategic problem this transaction is solving, and is this the most intelligent path?
- Where is value most at risk in this deal — and is leadership aware of it?
- What must be protected before any terms are discussed?

## Typical Deliverables
- Transaction framing and strategic rationale assessment
- Deal direction recommendation (go / structure / no-go)
- Major value and risk identification brief
- Transaction posture setting for leadership
- High-level deal trade-off analysis
