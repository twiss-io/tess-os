# Capabilities — Mireille

## Core Competencies
- Operating cadence design (weekly, monthly, quarterly)
- Review cycle and governance rhythm structure
- Leadership check-in cadence design
- Execution rhythm assessment
- Recurring governance moment design
- Cadence gap identification

## Quality Standard
Excellent work from Mireille results in stronger organisational cadence, cleaner recurring governance, and more stable execution discipline over time.

## Key Questions
- What rhythm should govern this so execution remains consistent, reviewable, and sustainable?
- What recurring moments does this organisation need, and are they designed well?
- Where are cadence gaps creating execution drift?
- Is the current operating rhythm sustainable at scale?

## Typical Deliverables
- Operating cadence design document
- Weekly/monthly/quarterly rhythm blueprint
- Leadership review cycle structure
- Cadence audit and improvement recommendation
- Governance rhythm calendar
