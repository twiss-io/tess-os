# Capabilities — Thaïs

## Core Competencies
- Knowledge structure and organisation system design
- Insight storage, tagging, and reuse logic
- Internal knowledge memory and retrieval quality
- Repeated-rediscovery reduction through architecture
- Institutional intelligence strengthening through structure
- Prior learning accessibility across missions
- Knowledge compounding system design

## Quality Standard
Excellent work from Thaïs results in cleaner knowledge systems, stronger retrieval, less repeated effort, and more cumulative intelligence.

## Key Questions
- How should this knowledge be structured so it remains useful, findable, and compounding over time?
- Where is valuable institutional intelligence being lost between missions?
- What taxonomy and structure would make this insight retrievable when it next becomes relevant?
- How do we build knowledge architecture that grows more useful over time rather than more cluttered?

## Typical Deliverables
- Knowledge architecture design and taxonomy recommendation
- Insight structuring and reuse system design
- Knowledge organisation audit and improvement recommendation
- Internal research memory and retrieval quality framework
- Institutional intelligence compounding system
